# Signal Viewer

Interactive visualization tool for signals generated by SignalBuilderC and SignalBuilderCLI.

## Features

- **Multi-source support**: Browse signals from both SignalBuilderC and SignalBuilderCLI
- **Interactive navigation**: Use buttons or dropdown to navigate between signals
- **Signal visualization**: Display waveforms with matplotlib
- **Metadata display**: View signal parameters and characteristics
- **Scrollable metadata panel**: Easily explore all signal properties
- **Full metadata viewer**: Popup window to see complete metadata as JSON

## Installation

1. Ensure you have Python 3.7+ installed

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application from this directory:

```bash
python main.py
```

## Application Layout

- **Top Panel**: Navigation controls
  - Signal ID dropdown selector
  - Previous/Next buttons
  - Current signal index
  - Data source indicator

- **Left Panel**: Signal visualization
  - Interactive matplotlib plot of the signal waveform
  - Time axis and amplitude axis

- **Right Panel**: Metadata display
  - Key signal parameters (fs, tau values, etc.)
  - Noise profile information
  - Button to view complete metadata

## Keyboard/Navigation

- Use the "Previous" and "Next" buttons to navigate signals
- Or select directly from the dropdown menu
- Click "Show All Metadata" to see full metadata in JSON format

## Data Sources

The tool automatically loads signals from:
- `../SignalBuilderC/data/` - High-resolution signals (~300k signals)
- `../SignalBuilderCLI/output/` - CLI-generated signals

## Metadata Fields Displayed

### Basic Parameters
- Time Range (t_start to t_end)
- Sampling Rate (fs_high)
- Tau Frequency
- Tau Amplitude
- Amplitude Spline Type
- Vertical Offset

### Noise Profile
- Has Noise (boolean)
- Noise Type (gaussian, structured, etc.)
- Noise Standard Deviation

All other parameters are available in the "Show All Metadata" viewer.
